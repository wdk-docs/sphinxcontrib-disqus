用法¶

本页介绍了如何开始使用扩展程序。

安装¶

用 pip 安装扩展:

pip install sphinxcontrib-disqus

配置¶

您必须在您的 conf.py 中指定您网站的 shortname 以及此扩展名。 这是一个示例文件，包含您需要执行的两项操作：

# General configuration.
author = 'Your Name Here'
copyright = '2015, Your Name Here'
exclude_patterns = ['_build']
extensions = ['sphinxcontrib.disqus']  # Add to this list.
master_doc = 'index'
project = 'my-cool-project'
release = '1.0'
version = '1.0'

# Options for extensions.
disqus_shortname = 'my-cool-project'  # Add this line to conf.py.

您必须在 extensions 中包含 sphinxcontrib.disqus ，并且必须使用 Disqus 管理页面中的短名称定义 disqus_shortname 。

在文档中使用¶

您可以在文档中指定其他选项。 在指定标 识符 时有两种选择

自动¶

扩展将使用文档标题(文档中的第一个标题)定义标识符。为此您只需指定：

.. disqus::

手册¶

要使用 disqus_identifier 选项手动设置文档的标识符，请指定：

.. disqus::
    :disqus_identifier: name_of_this_page
