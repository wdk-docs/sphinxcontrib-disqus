sphinxcontrib-disqus 2.0.19¶

在您的Sphinx文档中添加 Disqus 评论。

sphinxcontrib-disqus 提供了一个``disqus`` 指令，用于文档。 它会自动添加 Disqus 所需的 Javascript。本页底部显示了一个工作示例。

项目链接¶

  • Documentation: https://wdk-docs.github.io/sphinxcontrib-disqus

  • Source code: https://github.com/wdk-docs/sphinxcontrib-disqus

  • PyPI 主页: https://pypi.python.org/pypi/sphinxcontrib-disqus

注意事项¶

在本地生成Sphinx文档并浏览到 file:// URL时，不会加载注释。 这是因为Disqus下载的第二批Javascript代码与浏览器用于访问您所在页面的相同协议相匹配(在这种情况下，它是 file://)。 出于安全原因，Javascript不会从 file:// 加载，只能从 http:// 或 https:// 加载。

在本地编写文档时的一个简单的解决方法(一旦将文档托管在Web服务器或服务上，例如 阅读文档)，这不是问题”您的HTML文件通过简单的Web服务器。 在构建HTML文件后，您可以轻松运行这些命令(在OS X上测试)：

  1. cd docs/_build/html

  2. python -m SimpleHTTPServer 8080

  3. 浏览到 http://localhost:8080/
